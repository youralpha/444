# Сборка APK для Android

Поскольку на Android нельзя просто запустить локальный Flask-сервер, эта "мобильная" версия приложения работает полностью автономно (offline). Все данные сохраняются в `localStorage` вашего телефона и не стираются (если только вы сами не очистите данные приложения в настройках Android).

Для упаковки этих HTML-файлов в нативный Android `.apk` файл используется **Capacitor**.

## Требования:
1. Установленный **Node.js** (версия 18+).
2. Установленная **Android Studio** (для финальной компиляции APK).

## Инструкция по сборке

1. Откройте терминал (командную строку) в папке `mobile_app`.

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Инициализируйте проект Android:
   ```bash
   npx cap add android
   ```

4. Синхронизируйте HTML-файлы с Android-проектом:
   ```bash
   npx cap sync
   ```

5. Откройте проект в Android Studio:
   ```bash
   npx cap open android
   ```
   *(Если эта команда не сработала, просто откройте Android Studio, выберите "Open" и укажите папку `mobile_app/android`).*

6. В Android Studio дождитесь окончания синхронизации Gradle (внизу экрана будет ползти полоса загрузки).

7. Для создания APK-файла:
   - В верхнем меню Android Studio нажмите **Build** -> **Build Bundle(s) / APK(s)** -> **Build APK(s)**.
   - Когда сборка завершится, в правом нижнем углу появится уведомление со ссылкой `locate`. Нажмите на нее.
   - В открывшейся папке будет лежать ваш `app-debug.apk`.

Вы можете скинуть этот файл на свой телефон и установить его!
