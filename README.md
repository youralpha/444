# ПЕРИМЕТР v4.0

Это нативное десктопное приложение, объединяющее три психологических модуля (Оперативный Дашборд Периметр, Трекер КПТ Протокола, и Таймер Стрессоустойчивости/Базы) в единую удобную систему.

Приложение написано на языке **Rust** с использованием фреймворка **Dioxus**, использует встроенную базу данных **SQLite** (`rusqlite`) для локального и безопасного хранения прогресса, а также полностью кастомную, независимую тему стилей `tactical` (без внешних зависимостей типа Tailwind).

## 1. Системные требования

Для запуска и компиляции приложения необходим установленный язык Rust (Cargo).
Для работы Dioxus Webview на платформе **Linux** требуются следующие системные библиотеки:

```bash
sudo apt-get update
sudo apt-get install -y libwebkit2gtk-4.1-dev libgtk-3-dev libayatana-appindicator3-dev libxdo-dev
```

*Для Windows и macOS дополнительные системные графические библиотеки не требуются.*

## 2. Запуск в режиме разработки

Перейдите в директорию `perimetr` и выполните команду `cargo run`. Cargo автоматически скачает все крейты (зависимости), скомпилирует проект и откроет окно приложения:

```bash
cd perimetr
cargo run
```

## 3. Сборка для Production (Релиз)

Чтобы скомпилировать быстрый, автономный и оптимизированный исполняемый файл без дебаггера, выполните:

```bash
cd perimetr
cargo build --release
```

После компиляции исполняемый файл приложения можно будет найти в директории `perimetr/target/release/perimetr` (или `.exe` для Windows).

## 4. Локальное хранилище данных

Приложение 100% автономно и не требует подключения к интернету. Все данные записываются в файл `perimetr.db`, который автоматически создается рядом с исполняемым файлом. Этот файл содержит прогресс по:
- Агентурной сети и задачам Периметра.
- Накопленному опыту (XP) и выполненным заданиям КПТ-протокола (и прогрессу чтения книги).
- Общему состоянию системы и счетчику "Агента 4444".
