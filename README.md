# Tactical Dashboard (Kotlin/Wasm Compose Multiplatform)

**Внимание: Приложение находится в процессе полномасштабной миграции с архитектуры HTML/JS (Vanilla + Tailwind) на Kotlin Multiplatform (Compose).**
Текущий коммит представляет собой **Фундамент** нового приложения: настроена сборка для WebAssembly (Wasm), реализована навигация и сверстаны базовые макеты всех экранов (Таймер, КПТ Протокол, Периметр) в едином "Тактическом" стиле.
Полная логика (циклы дыхания, графики, Drag & Drop агентурной сети) будет переноситься поэтапно в следующих обновлениях.

## Архитектура
- **Язык:** Kotlin 2.0.0
- **Фреймворк UI:** Compose Multiplatform
- **Целевая платформа:** WasmJs (запускается прямо в браузере)
- **Хранение данных:** Взаимодействие с `localStorage` браузера через JS-Interop.

В будущем эта единая кодовая база позволит собирать нативные приложения для Windows (Desktop) и Android без изменения бизнес-логики и UI-кода.

## Сборка и запуск локально
Для запуска требуется установленный **JDK 17** (или выше).

1. Склонируйте репозиторий.
2. Откройте терминал в корневой папке.
3. Запустите локальный сервер разработки с горячей перезагрузкой:
   ```bash
   ./gradlew wasmJsBrowserDevelopmentRun
   ```
4. Откройте бразуер по адресу (обычно `http://localhost:8080`), который будет указан в терминале.

## Структура папок
Все основные исходники находятся в `composeApp/src/commonMain/kotlin/org/tactical/dashboard/`:
- `theme/` — Цветовые схемы, типографика (Dark Tactical Theme).
- `ui/` — Композитные функции экранов (Главное меню, База, КПТ, Периметр).
- `data/` — Логика хранения и работы с данными.
