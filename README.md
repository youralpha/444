# Интегрированное веб-приложение (Таймер, КПТ Протокол, Периметр)

Это приложение объединяет три HTML-интерфейса в один Flask-сервер и использует базу данных SQLite для сохранения данных, чтобы они не терялись после закрытия браузера.

## Требования
- Python 3.7+
- pip

## Установка и запуск

1. Склонируйте или скачайте этот репозиторий.
2. (Рекомендуется) Создайте виртуальное окружение и активируйте его.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   python app.py
   ```
   При запуске автоматически откроется нативное окно приложения. База данных `data.db` будет создана в папке профиля пользователя Windows: `C:\Users\<ВашеИмя>\.perimeter_app`.

   *(Если у вас проблемы с `pywebview`, приложение запустится просто как сервер, и вы сможете открыть его в браузере по адресу `http://127.0.0.1:5000`)*.

## Как создать .exe файл (для Windows)

1. Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`.
2. Запустите скрипт сборки:
   - Просто дважды кликните на файл `build.bat`
   - ИЛИ выполните команду в терминале: `pyinstaller --noconfirm --onedir --windowed --add-data "templates;templates" --name "Perimeter" app.py`
3. После успешной сборки в папке `dist/Perimeter/` появится готовый исполняемый файл `Perimeter.exe`. Эту папку (или сам экзешник, если собирать с флагом `--onefile`) можно переносить на другие компьютеры.

## Структура файлов
- `app.py` - Основной код сервера Flask.
- `requirements.txt` - Зависимости Python.
- `data.db` - Файл базы данных SQLite (появится после первого запуска).
- `templates/` - Папка с HTML-файлами.
  - `index.html` - Главное меню.
  - `BASE.html` - Таймер дыхания.
  - `KPT.html` - КПТ Протокол (с сохранением в БД).
  - `PERIMETR.html` - ПЕРИМЕТР v4.0 (с сохранением в БД).
