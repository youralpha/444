# Оперативный Дашборд (ПЕРИМЕТР / КПТ / ТАЙМЕР)

Единое приложение, переписанное на связку **React.js + Tailwind CSS** (Фронтенд) и **Flask + SQLite** (Бэкенд).
Оно сохраняет все данные локально на устройстве (в папке профиля пользователя `~/.perimeter_app/data.db`), обеспечивая полную автономность и безопасность данных.

## Требования
- Node.js (v18+)
- Python 3.7+

## 1. Запуск Бэкенда (Python/Flask)
1. Откройте терминал в корневой папке.
2. (Рекомендуется) Создайте и активируйте виртуальное окружение:
3. Установите зависимости:
   `pip install -r requirements.txt`
4. Запустите сервер:
   `python app.py`
   Сервер запустится на `http://127.0.0.1:5000`.

## 2. Запуск Фронтенда (React/Vite)
1. Откройте **новый** терминал и перейдите в папку `frontend`:
   `cd frontend`
2. Установите JS-зависимости:
   `npm install`
3. Запустите сервер разработки:
   `npm run dev`
4. Откройте в браузере адрес, указанный Vite (обычно `http://localhost:5173`). Все запросы к `/api/*` будут автоматически проксироваться на Flask.

## Архитектура
- **frontend/src/** — исходный код React компонентов (Таймер, КПТ, Периметр) в темном "тактическом" стиле.
- **app.py** — легковесный REST API на Flask, обслуживающий сохранение и получение JSON состояния в SQLite.
